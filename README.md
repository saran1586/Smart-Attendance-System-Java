Attendance System

This is a Java-based Smart Attendance System developed using JDBC and MySQL.

 Features
- Mark attendance
- View attendance records
- Stores attendance data in MySQL database

Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

 How It Works
The system allows users to mark attendance and stores the details in a database.
It helps to manage attendance digitally instead of manual registers.

How to Run
1. Download or clone the project from GitHub
2. Import the project into Eclipse
3. Configure MySQL database
4. Run the MainApp.java file

- Web-based interface
